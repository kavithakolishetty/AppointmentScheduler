
# CalenderBooking


What is the Calender Booking Project?
=====================

The goal of this console application is implement the logic to block 30 mins slots based on availabilty.


## How to use:
- You will need the latest Visual Studio 2022 and .NET 7.0
- ***Please check if you have installed the same runtime version (SDK) used in project***
- Install EntityFramwork 6.4.4
- Install Microsoft.EntityFrameworkCore 6.0.0
- Install Microsoft.Extensions.DependencyInjection 6.0.0
- Install Moq 4.20.70
- Install Nuint3TestAdapter 4.4.2
- Install Sql Express Database
- Run TableCreation_Script.sql to create database and Table
Replace connectionString value in App.config

## Technologies implemented:

- .NET Core  
- .NET Core Native DI
-  MOQ Unit tests


## Architecture:

- Full architecture with responsibility separation concerns, SOLID and Clean Code
- Dependeny Injection
- Repository
